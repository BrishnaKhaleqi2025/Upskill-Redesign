Project
Overview

This project focused on redesigning the reporter news page that is put only RIT students. It included having to rebrand the reporter but also to redesign and animate prototypes using after effects.

The problem

01

No definitive style

The current Reporter page falls flat when it comes to style so it was my goal to create a trendy new style for consistency throughout the site

03

Scanability

The lack of hierarchy within the layout makes it difficult to quickly browse content the user

02

Category Emphasis

Currently categories have their own color however it appears hidden so finding a better way to emphasize each category

04

Reducing Clutter

The layout is somewhat scattered with no distinct order which can be overwhelming when scanning

Desktop & Mobile Sketching

Sketches included many alternative ways to view not only the home page but also viewing an article on the page. I also wanted to focus on the category that each article fit into

Complemetary

I decided to go with the complementary moodboard because I thought the peach would be a great way to use the color in terms of rebranding but also has a youthful vibe which would target the RIT community the best
